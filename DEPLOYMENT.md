# Budgeting App - Deployment

## Live App
**URL**: https://easy-budget.expo.app/

### Build
```bash
npx expo export --platform web
```

### Building if you changed the .env file
```bash
npx expo export --platform web --clear
```

### Deploy
```bash
eas deploy
```
